# PromptCraft Pro 🚀
**Architect Intelligence, Lead Transformation.**

A cinematic, high-performance learning platform for IT Managers to master Prompt Engineering.

## 🛠 Tech Stack
- **Frontend**: Next.js 14, Tailwind CSS, Framer Motion
- **Backend**: Node.js, Express, Socket.io
- **Database**: PostgreSQL
- **Auth**: NextAuth.js

## 🌟 Key Features
- **Cinematic UI**: Glassmorphism and "Deep Space" aesthetics.
- **AI Sandbox**: Real-time prompt testing with a built-in Strength Meter.
- **Admin Dashboard**: Team progress tracking for Enterprise leaders.
- **Hybrid Learning**: Logic for 1:1 and Group coaching sessions.

## 🚀 Quick Start
1. \git clone https://github.com/Apolloat2022/Prompt-Engineering_Courses.git\
2. \cd frontend && npm install && npm run dev\
3. \cd backend && npm install && npm start\
