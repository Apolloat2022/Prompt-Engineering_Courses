# PromptCraft Pro 🚀
**Architect Intelligence, Lead Transformation.**

A cinematic, high-performance learning platform for IT Professionals to master Prompt Engineering and Agentic Workflows.

## 🛠 Tech Stack
- **Framework**: Next.js 14 (App Router)
- **Styling**: Tailwind CSS, Custom Animations
- **State Management**: React Hooks + LocalStorage Persistence
- **Auth**: NextAuth.js (Client-side integration)
- **Deployment**: Serverless Ready (Vercel/Netlify compatible)

## 🌟 Key Features
- **Cinematic UI**: Glassmorphism and "Deep Space" aesthetics designed for immersion.
- **Dual Curriculum**: 
  - **Level 1**: AI Communication Fundamentals (Prompting, CoT, Zero-shot).
  - **Level 2**: Agentic Workflows (Autonomous Loops, RAG, Multi-Agent Systems).
- **AI Sandbox**: Real-time prompt scoring engine with heuristic feedback.
- **"Buildfolio" Certificate**: Dynamic, print-ready certification system with professional branding.
- **User Profile**: Gamified stats tracking (XP, Rank, Module History).

## 🚀 Quick Start
This project is a single-stack Next.js application.

1. **Install Dependencies**
   ```bash
   npm install
   ```

2. **Run Development Server**
   ```bash
   npm run dev
   ```

3. **Open Application**
   Visit `http://localhost:3000` to start your journey.

## 📂 Project Structure
- `src/app`: App Router pages and layouts.
- `src/components`: Reusable UI components.
- `src/data`: Curriculum content and quiz data.
- `src/hooks`: Custom hooks (useProgress) for state persistence.
